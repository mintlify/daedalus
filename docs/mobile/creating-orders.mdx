## Creating an order

### Navigate to your portfolio

Click on a trading account or portfolio option from the `Home` tab. There, you will see the `My Account` page.

<Frame>
  <img
    class="h-96 rounded"
    src="/images/mobile/orders/user-account.png"
    alt="User Account"
  />
</Frame>

### Add order

Click on the `Add Order` button on the bottom-right side of the page.

### Select asset type

Select whether you would like to add `Crypto` or `Stocks`

<Frame>
  <img
    class="h-96 rounded"
    src="/images/mobile/orders/asset-type.png"
    alt="Select asset type"
  />
</Frame>

### Select asset

<Frame>
  <img
    class="h-96 rounded"
    src="/images/mobile/orders/select-asset.png"
    alt="Select asset"
  />
</Frame>

### Select buy/sell

Select whether you would like to buy or sell using the top navigation buttons

<Frame>
  <div class="flex space-x-2">
    <img class="h-96 rounded" src="/images/mobile/orders/buy.png" alt="Buy" />
    <img class="h-96 rounded" src="/images/mobile/orders/sell.png" alt="Sell" />
  </div>
</Frame>

<Tip>

You can also directly buy or sell by going to the `Markets` tab and clicking
on an individual asset.

</Tip>

### Select order options

Choose the details of your purchase or sale. You can select options including

- `Order Type`
- `Time in Force`
- `Trading Pair`

<Frame>
  <img
    class="h-96 rounded"
    src="/images/mobile/orders/order-type.png"
    alt="Order type"
  />
</Frame>

### Select number of shares/dollars

Select whether you would like to make the transaction on the quantity of shares or the dollar amount.

<Frame>
  <div class="flex space-x-2">
    <img
      class="h-96 rounded"
      src="/images/mobile/orders/quantity-shares.png"
      alt="Shares"
    />
    <img
      class="h-96 rounded"
      src="/images/mobile/orders/amount-dollars.png"
      alt="Amount"
    />
  </div>
</Frame>

### Confirm the order

Confirm whether you would like to make the order

<Frame>
  <img
    class="h-96 rounded"
    src="/images/mobile/orders/place-order-confirmation.png"
    alt="Confirm order"
  />
</Frame>

And you are done! Now, you can see your order showcased in the `Orders` section of `My Account`.

<Frame>
  <img
    class="h-96 rounded"
    src="/images/mobile/orders/order-placed.png"
    alt="Order placed"
  />
</Frame>
